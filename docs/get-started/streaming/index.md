# 串流快速開始

Agent Development Kit (ADK) 讓你的 AI agent 能夠透過串流實現即時、互動式的體驗。這使你可以實現像是即時語音對話、即時工具使用，以及 agent 的持續更新等功能。

本頁提供 Python 與 Java 兩種 ADK 的串流功能快速開始範例，協助你快速上手。
