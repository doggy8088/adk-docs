# 串流快速開始

Agent Development Kit (ADK)（ADK）讓你的 AI 代理（agent）能夠透過串流實現即時、互動式體驗。這使你可以實作如即時語音對話、即時工具使用，以及 agent 持續回報更新等功能。

本頁提供 Python 及 Java ADK 串流功能的快速開始範例，協助你快速上手。
