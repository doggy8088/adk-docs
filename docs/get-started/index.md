# 開始使用

Agent Development Kit (ADK) 旨在協助開發者打造、管理、評估與部署 AI 驅動的 agent。它提供一個強大且靈活的環境，讓你可以建立具備處理複雜任務與工作流程能力的對話式與非對話式 agent。
