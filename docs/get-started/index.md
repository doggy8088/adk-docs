# 開始使用

Agent Development Kit (ADK)（ADK）旨在協助開發者打造、管理、評估與部署 AI 驅動的代理（agent）。它提供一個強大且靈活的環境，可用於建立對話式與非對話式的 agent，能夠處理複雜的任務與工作流程。
